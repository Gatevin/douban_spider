Usage:
1. install golang environment
2. create file directory src/
3. in src/ , git clone
4. make release according to your os:
```
# windows
make release_windows
# macos
make release_macos
# linux
make_release_linux
```

5. run
```
# windows
./bin/douban_spider.exe

# macos
./bin/douban_spider

# linux
./bin/douban_spider
